# Records-System 💧

istema simples em Python para gerenciamento de protocolos, utilizando persistência de dados em arquivos CSV.

## ✨ Funcionalidades
- Cadastro de protocolos com data e hora automática.
- Organização visual por data no arquivo CSV.
- Exclusão de registros com trava de segurança para separadores.
- Visualização tabelada no terminal.

## 🛠️ Tecnologias
- Python 3.x
- Bibliotecas: `csv`, `os`, `datetime`
