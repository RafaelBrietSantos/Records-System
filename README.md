#  Records System 💧 (Em Desenvolvimento 🚧)

Este projeto é um sistema de gerenciamento de protocolos em Python. Ele está sendo construído para facilitar o registro e a organização de atendimentos.

> **Status do Projeto:** 🛠️ Em construção.

## ✅ O que já funciona:
- [x] Menu interativo.
- [x] Cadastro com data/hora automática.
- [x] Sistema de separadores por dia no CSV.
- [x] Visualização organizada no terminal.
- [x] Exclusão de protocolos com trava de segurança.

## 🚀 Próximas Implementações:
- [ ] **Função de Editar:** Alterar o status de um protocolo já existente.
- [ ] **Busca por Protocolo:** Filtrar um atendimento específico sem precisar listar tudo.
- [ ] **Exportação de Relatórios:** Criar um resumo dos atendimentos do dia.
- [ ] **Interface Visual:** Talvez evoluir do terminal para uma janelinha (GUI).
